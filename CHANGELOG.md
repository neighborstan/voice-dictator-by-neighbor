## [Unreleased]

### Добавлено

- Полный цикл диктовки: нажал хоткей → говоришь → нажал снова → текст вставляется в активное поле
    - Оркестрация pipeline: запись → транскрипция (OpenAI STT) → улучшение текста → вставка
    - Safety timeout: запись автоматически останавливается при превышении максимальной длительности
    - Cancel: повторное нажатие хоткея во время обработки отменяет текущую задачу и возвращает приложение в Idle
- Result window: окно-запаска с транскрибированным текстом, когда вставка через clipboard невозможна
    - Кнопки Copy (нативный clipboard) и Close
    - Авто-закрытие через 10 секунд (по умолчанию включено)
    - Всегда поверх других окон

### Исправлено

- PTT-режим: нажатие хоткея во время Transcribing/Enhancing/Pasting теперь корректно отменяет задачу (раньше игнорировалось)
- Result window: кнопка Copy работает через системный clipboard вместо browser clipboard API
- Ошибки при показе result window теперь попадают в лог (не игнорируются)
