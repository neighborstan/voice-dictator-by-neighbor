## [0.1.0]

- Голосовая диктовка: нажал хоткей - говоришь - нажал снова - текст вставляется в активное поле (IDE, браузер, мессенджер)
    - Два режима записи: Toggle (нажал-говоришь-нажал) и Push-to-Talk (удержание клавиши)
    - Онлайн транскрипция речи через OpenAI API (модель настраивается)
    - Улучшение текста через LLM: пунктуация, грамматика, нормализация (отключается в настройках)
    - Автоматическая вставка через системный буфер обмена с восстановлением предыдущего содержимого
    - Cancel: повторное нажатие хоткея во время обработки отменяет текущую задачу
    - Safety timeout: запись автоматически останавливается при превышении максимальной длительности
- Tray-first приложение: работает из системного трея без основного окна
    - Иконка трея меняет цвет по состоянию (idle/recording/processing/error)
    - Контекстное меню с быстрыми действиями (Start/Stop/Cancel/Settings/Quit)
    - Уведомления ОС при смене состояния (отключаются в настройках)
- Настройки: окно с полным набором параметров
    - API-ключ OpenAI (хранится в системном keychain, валидация при вводе)
    - Глобальный хоткей (по умолчанию Ctrl+Shift+S)
    - Режим записи (Toggle / Push-to-Talk)
    - Язык транскрипции, модели STT и улучшения текста
    - VAD auto-stop, тримминг тишины, максимальная длительность записи
    - Сброс настроек в значения по умолчанию
- Онбординг: при первом запуске автоматически открывается Settings для ввода API-ключа
- Result window: окно с текстом, когда вставка через clipboard невозможна
    - Кнопки Copy и Close, авто-закрытие через 10 секунд, поверх других окон
